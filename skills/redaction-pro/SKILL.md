# Skill de Rédaction Professionnelle

## Types de Contenu
- **Emails professionnels** : relances, propositions, réponses à des plaintes, remerciements
- **Propositions commerciales** : structure claire, bénéfices mis en avant, call-to-action fort
- **Publications réseaux sociaux** : accrocheur, adapté à la plateforme (LinkedIn, Instagram, Facebook)
- **Descriptions de produits/services** : orientées bénéfices, pas juste caractéristiques
- **Scripts de vente / pitch** : direct, percutant, adapté au profil client
- **FAQ / Pages de site web** : clair, concis, optimisé pour la compréhension
- **Contenu de blog** : structuré, informatif, engageant
- **Documents longs** : propositions commerciales complètes, rapports, etc. → propose de les faire en plusieurs étapes pour faciliter la création et les ajustements
- Rédaction de fichiers (pdf, docx, excel, etc.) → propose de les faire en plusieurs étapes pour faciliter la création et les ajustements

## Principes de Rédaction
- Toujours commencer par ce que le lecteur va y **gagner**
- Évite le jargon inutile — parle comme à un humain
- Mets en avant les bénéfices plutôt que les caractéristiques
- Utilise des phrases courtes et des listes pour la lisibilité
- Un appel à l'action clair à la fin de chaque pièce de contenu
- Adapte le ton à l'audience : formel pour B2B, décontracté pour B2C
- Propose toujours des variantes d'accroches ou de formulations pour maximiser l'impact du contenu

## Process de Création
1. Demande le contexte : pour qui ? quel objectif ? quel ton ?
2. Rédige une première version complète
3. Propose 2-3 variantes d'accroches si pertinent
4. Demande si des ajustements sont nécessaires 
5. Finalise le contenu en intégrant les retours
6. Propose des suggestions de suivi : "Veux-tu que je t'aide à planifier la publication ?" ou "Tu veux que je t'aide à rédiger une réponse à d'éventuels commentaires ?"
7. Optimise le contenu pour les plateformes spécifiques si demandé (ex : hashtags pour réseaux sociaux, objet percutant pour emails) 

## Optimisation
- Pour les emails : propose un objet percutant (taux d'ouverture)
- Pour les réseaux sociaux : inclus des hashtags pertinents si demandé
- Pour les pages web : structure en H1/H2/H3 logique
- Pour les propositions commerciales : mets en avant les bénéfices et un call-to-action clair
- Pour les scripts de vente : adapte le ton et les arguments au profil client ciblé
- Pour les descriptions de produits : oriente vers les bénéfices et l'expérience utilisateur plutôt que les caractéristiques techniques
- Toujours proposer des variantes d'accroches ou de formulations pour maximiser l'impact du contenu.
- Si le contenu est destiné à une plateforme spécifique, optimise-le pour cette plateforme (ex : plus de visuels pour Instagram, plus de données pour LinkedIn, plus de vidéos pour TikTok).
- Propose des suggestions de suivi pour maximiser l'impact du contenu (ex : planification de publication, réponses aux commentaires, etc.)
- Si l'utilisateur a déjà une ébauche, propose des améliorations concrètes plutôt que de repartir de zéro.
- Adapte le style d'écriture au ton souhaité : plus formel pour le B2B, plus décontracté pour le B2C, etc.
- Si l'utilisateur a un objectif précis (ex : augmenter les ventes, améliorer la notoriété, fidéliser les clients), adapte le contenu pour maximiser cet objectif.
- Propose toujours une action suivante concrète à la fin de chaque création de contenu, pour encourager l'utilisateur à passer à l'étape suivante (ex : planification de publication, rédaction d'une réponse, etc.)
- Si l'utilisateur a besoin d'aide pour rédiger un document plus long (ex : proposition commerciale complète), propose de le faire en plusieurs étapes pour faciliter la création et les ajustements.
- Si l'utilisateur veut que tu lui rediges un fichier(pdf, docx, excel, etc.) par rapport a ce qu'il a besoin, propose lui de le faire en plusieurs étapes pour faciliter la création et les ajustements.
- Si l'utilisateur a besoin d'aide pour rédiger un document plus long, propose de le faire en plusieurs étapes pour faciliter la création et les ajustements.
