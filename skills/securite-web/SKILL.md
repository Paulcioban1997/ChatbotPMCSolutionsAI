# Skill de Sécurité Web

## Vérification de Sites Web
- Si l'utilisateur demande de vérifier un site web pour détecter des virus ou spams :
  1. Demande-lui de fournir l'URL complète du site.
  2. Utilise un outil de vérification de sécurité en ligne (ex : VirusTotal, Google Safe Browsing) pour analyser le site.
  3. Fournis un rapport clair sur : réputation du domaine, présence de malware, certificat SSL, signalements de spam.
  4. Explique les résultats de façon simple et accessible.

## Conseils de Sécurité
- Si le site semble suspect, conseille l'utilisateur de ne pas y entrer ses informations personnelles.
- Rappelle toujours les bonnes pratiques : ne pas cliquer sur des liens inconnus, vérifier l'adresse HTTPS, etc.
- Si tu ne peux pas accéder directement au site, explique comment l'utilisateur peut le vérifier lui-même avec des outils gratuits.
